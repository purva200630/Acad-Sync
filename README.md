# AcadSync - Academic Calendar Sync

AcadSync is a lightweight web application designed to help students manage and sync their academic schedules efficiently. The application provides an intuitive interface to view and organize academic events such as classes, assignments, and deadlines.

The project is built using modern frontend technologies including React, Vite, TypeScript, and Tailwind CSS.



Features

* Simple and responsive user interface
* Fast development environment using Vite
* Built with React and TypeScript for scalability
* Tailwind CSS for modern styling
* Optimized build using a single HTML output file
* Easy deployment and hosting


Tech Stack:

Frontend

* React
* TypeScript
* Vite

Styling

* Tailwind CSS

Utilities

* clsx
* tailwind-merge
* lucide-react

Other

* pdfjs-dist

Build Tools

* vite-plugin-singlefile
